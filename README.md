# Calculator GUI

A simple calculator GUI application built with Python and Tkinter.

## Features

- Basic arithmetic operations (+, -, ×, ÷)
- Square (x²) and square root (√x) functions
- Keyboard support for number and operator input
- Clean and intuitive user interface

## Requirements

- Python 3.x
- Tkinter (usually included with Python)

## Installation

1. Clone the repository:
```bash
git clone https://github.com/soggy8/calculator-gui.git
cd calculator-gui
```

2. Run the application:
```bash
python app.py
```

## Usage

### Mouse Input
- Click on number buttons (0-9) to enter digits
- Click on operator buttons (+, -, ×, ÷) to perform operations
- Click `=` to evaluate the expression
- Click `C` to clear the display
- Click `x²` to square the current number
- Click `√x` to calculate the square root

### Keyboard Input
- Use number keys (0-9) to enter digits
- Use operator keys (+, -, *, /) for operations
- Press `Enter` to evaluate the expression
